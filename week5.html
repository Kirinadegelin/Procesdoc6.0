Github en Netlify

We hebben allemaal een account moeten aanmaken in github en netlify, github had ik al eens van gehoord maar ik wist helemaal niet wat het was.

De accounts aanmaken lukte helemaal, de feedback op de figma schermen was nogal hard, ik schrok ervan hoe netjes en fatsoenlijk een mid-prototype al moest zijn.

Het uploaden van mijn html file ging vlotjes, ik heb de andere wat geholpen omdat ze wat problemen hadden of het niet snapte.
